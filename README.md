# 🤜 АНИКИ — ИИ-ассистент Билли Херрингтона

> *"Are you ready? Let's go!"*

Полностью локальный ИИ-ассистент на основе персонажа Билли Херрингтона. Работает **без интернета**, **без подписок**, **без серверов**. Твой личный «Аники» — дружелюбный, мотивирующий, и немного гачи.

| 💬 **ИИ-чат** | Разговаривает на русском с гачимучи-вставками |

| 🔔 **Трей** | Живёт в системном трее, не мешает |

## 🚀 Быстрый старт

### Шаг 1: Установи Python

Скачай [Python 3.11+](https://python.org) и установи.  
**⚠️ Важно:** поставь галочку **"Add Python to PATH"**

### Шаг 2: Установи зависимости

```
Дважды кликни setup.bat
```

### Шаг 3: Установи Ollama (локальный ИИ)

1. Скачай [Ollama для Windows](https://ollama.com)
2. Установи
3. Открой cmd и выполни:
```cmd
ollama run mistral
```
*(Первый запуск скачает ~4GB — это нормально, один раз)*

### Шаг 4: Запусти Аники!

```
Дважды кликни aniki.bat
```

или дождись автонастройки при первом запуске.

## 🛠️ Технологии

| Компонент | Технология |
|-----------|-----------|
| 🧠 ИИ-мозг | **Ollama** + **Mistral 7B** (локально) |
| 🎤 Распознавание речи | **faster-whisper** (офлайн, Whisper small) |
| 🔊 Синтез речи | **Silero TTS v4** (русский, нейросеть) |
| 🖥️ Интерфейс | **PyQt6** (темная тема, системный трей) |
| 💾 База данных | **SQLite** (память, напоминания) |
| 🪟 Windows API | **pycaw** (звук), **pywin32** (система) |
| 📦 Упаковка | **PyInstaller** (.exe) |

## ⚙️ Настройки напоминаний

В интерфейсе во вкладке **Настройки** можно изменить:
- Интервал напоминания о воде (по умолчанию: 60 мин)
- Интервал напоминания о перерыве (по умолчанию: 90 мин)
- Порог игрового времени для предупреждения (по умолчанию: 120 мин)

---

## 🔧 Устранение неполадок

**"Ollama не запущен"** — запусти `ollama serve` в cmd  
**"Модель не найдена"** — запусти `ollama run mistral`  
**Нет голоса** — убедись что torch и sounddevice установлены  
**Нет распознавания речи** — проверь микрофон в настройках Windows
